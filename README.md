## Quick Start & Documentation
Local development server:
```
yarn dev
http://localhost:3000/en
```

Local SSR run:
```
yarn build && yarn start:dev
http://localhost:3000/en
```

Production build and deployment:  
```
yarn deploy  
https://anirbmuk.appspot.com/en  
```
